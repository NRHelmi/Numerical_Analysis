* Cette ligne de commande permet la compilation des c_fichiers sources contenants
 la bibliotheque math.h : gcc c_fichier_source -o fihcier_objet -lm

* La fonction mem_disp() (pour dire mémoire disponible) gére d'une maniére automatique
 la taille des matrices en fonction de la mémoire disponible, en faite vous devez executer
 votre programme en mode super user afin de bénéficier d'une taille maximale et pour eviter
 tout probleme d'allocation mémoire :
  sudo ./fichier_objet
